lesscss:compile

Full name:

biz.gabrys.maven.plugins:lesscss-maven-plugin:1.2.0-SNAPSHOT:compile

Description:

Compiles Less files to CSS stylesheets using extended version of the LessCSS Compiler.

Attributes:

  • Requires a Maven project to be executed.
  • Requires dependency resolution of artifacts in scope: runtime+system.
  • Since version: 1.0.
  • Binds by default to the lifecycle phase: generate-sources.
